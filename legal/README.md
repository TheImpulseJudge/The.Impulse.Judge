# Legal Documentation

This folder contains legal implementation guides and technical documentation for The Impulse Judge project.

## Contents

### ARBITRATION_NOTICE_IMPLEMENTATION.md
Technical implementation guide for the arbitration agreement notice UX. This document explains:
- How to implement the conspicuous arbitration notice (Section 14 of Terms of Use)
- Code examples for Chrome and Firefox extensions
- Legal compliance requirements and best practices
- Testing checklist and enforcement considerations

**Target audience:** Developers implementing the extension's terms acceptance flow

---

## Not a Lawyer?

These documents are **technical implementation guides**, not legal advice. They explain how to implement legal requirements that have already been drafted in the Terms of Use and Privacy Policy.

**If you need legal advice:** Consult a licensed attorney in your jurisdiction.

**If you need implementation help:** Read the guides in this folder or open a GitHub issue with tag `legal-compliance`.
